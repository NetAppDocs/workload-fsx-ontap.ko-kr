= NetApp Workload Factory에서 리소스 비용을 추적하고 보고서를 받아보세요.
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
NetApp Workload Factory 콘솔에서 FSx for ONTAP 파일 시스템의 비용 및 사용량을 통합된 보기로 추적하세요. 비용 데이터를 통해 예산을 관리하고 리소스를 효과적으로 최적화할 수 있습니다. Workload Factory는 또한 FSx for ONTAP 파일 시스템 리소스에 대한 비용 보고서를 생성하여 이해관계자와 공유할 수 있습니다.



[[about-this-task]]
== 이 작업에 대해

Workload Factory는 AWS Cost Explorer에서 비용 데이터를 가져와 FSx for ONTAP 파일 시스템의 비용 및 사용량을 추적합니다. 추적되는 비용 세부 정보는 사용 유형, 사용 SKU, 배포 유형, 사용량 및 총 비용입니다.

고유한 리소스의 경우, Workload Factory는 실제 성능 계층 사용량, 프로비저닝되었지만 사용되지 않은 공간에 할당된 성능 계층 비용의 비례 배분, 용량별로 분배된 성능 할당량, 그리고 용량 풀 사용량을 종합하여 비용을 계산합니다. 각 구성 요소는 해당 차원의 단위당 가격으로 곱해지고, 그 결과들을 모두 더하여 각 리소스의 총비용을 산출합니다.

비용 대시보드에서 AWS 계정, 자격 증명, 하나 이상의 리전 및 기간별로 비용 및 사용량 데이터를 필터링하여 FSx for ONTAP 파일 시스템 사용량을 파악할 수 있습니다. 데이터는 항상 월 단위로 표시됩니다.

저희는 다음 권한을 사용하여 AWS Cost Explorer에서 FSx for ONTAP 파일 시스템 리소스에 대한 비용 및 사용량 데이터를 가져옵니다. `ce:GetCostAndUsage`and `ce:GetTags` AWS 비용 API와 자체 추정치를 사용하여 총 비용을 계산합니다. `

AWS 할당 태그 키와 태그 키 값은 특정 비용을 정확하게 파악하기 위한 추가적인 필터링 기능을 제공합니다. 태그는 키와 값으로 구성됩니다. 태그 키와 태그 키 값으로 필터링하려면 FSx for ONTAP 파일 시스템에 태그를 할당하세요. link:https://docs.aws.amazon.com/awsaccountbilling/latest/aboutv2/cost-alloc-tags.html["AWS 비용 할당 태그에 대해 자세히 알아보세요."^]

비용 대시보드에 표시되는 비용 데이터 외에도 FSx for ONTAP 파일 시스템 리소스에 대한 비용 보고서를 생성할 수 있습니다.  비용 보고서에는 특정 파일 시스템 및 볼륨에 대한 정보가 표시됩니다.  이 보고서는 다운로드 가능한 PDF 파일 형식으로 제공되며, 이해 관계자와 공유할 수 있습니다.



[[track-costs]]
== 비용 추적

Workload Factory 콘솔의 비용 대시보드에 표시된 비용은 24시간마다 업데이트되는 AWS Cost Explorer의 데이터를 기반으로 합니다.

.시작하기 전에
다음 요구 사항을 충족하는지 확인하십시오.

* FSx for ONTAP 비용을 추적하려면 Workload Factory에서 link:https://docs.netapp.com/us-en/workload-setup-admin/add-credentials.html["_view, planning, and analysis_ 권한 정책으로 자격 증명을 부여합니다."^]해야 합니다.
* AWS 할당 태그로 필터링하려면 FSx for ONTAP 파일 시스템에 태그가 할당되어 있는지 확인하십시오.


.단계
. 중 하나를 사용하여 link:https://docs.netapp.com/us-en/workload-setup-admin/console-experiences.html["콘솔 환경"^]로그인합니다.
. 메뉴를 선택하세요 image:workloads-menu-icon.png["햄버거 메뉴 아이콘은 스토리지, EDA, AI, 데이터베이스, VMware 및 관리와 같은 워크로드로 이동하는 데 사용됩니다."] 그런 다음 *저장소*를 선택하세요.
. 저장소 메뉴에서 *비용*을 선택합니다.
. 비용 대시보드에서 다음 정보를 입력하여 FSx for ONTAP 리소스에 대한 비용 및 용량 데이터를 필터링하세요.
+
.. *AWS 계정*: 비용 데이터를 보려는 계정을 선택하세요.
.. *자격 증명*: _보기, 계획 및 분석_ 권한이 있는 자격 증명을 선택합니다.
.. *지역*: 비용 데이터를 보려는 AWS 지역을 선택하세요.
.. *비용 데이터 범위*: 비용 데이터를 볼 시간 범위를 선택하세요.
.. *할당 태그 키*: (선택 사항) AWS 태그와 연결된 모든 FSx for ONTAP 파일 시스템에 대한 자세한 비용 보고서를 생성하려면 할당 태그 키와 해당 값을 선택하십시오.
.. *할당 태그 값*: (선택 사항) AWS 태그와 연결된 모든 FSx for ONTAP 파일 시스템에 대한 자세한 비용 보고서를 생성하려면 할당 태그 키와 연결된 할당 태그 키 값을 선택하십시오.


. FSx for ONTAP 리소스에 대한 *비용 세부정보*를 확인하세요.
